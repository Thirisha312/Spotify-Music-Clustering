# Spotify Music Clustering

This project clusters Spotify songs based on their audio characteristics using Machine Learning.

## Features Used
- Danceability
- Energy
- Tempo
- Loudness
- Valence

## Techniques Used
- Data Preprocessing
- Feature Normalization
- KMeans Clustering
- PCA Visualization
- t-SNE Visualization

## Libraries
- pandas
- numpy
- scikit-learn
- matplotlib
- seaborn

## Output
Songs are grouped into clusters based on audio similarity to discover music listening patterns.
